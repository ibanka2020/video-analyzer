## 🔧 【重要】如何修复 Kimi / 豆包失效的问题

AI网站会**频繁更新**，导致插件的“输入框选择器”和“发送按钮选择器”**失效**。这会导致粘贴失败或无法自动发送。

**你需要学会自己查找最新的CSS选择器：**

1.  **打开AI网站**：例如，打开 Kimi (`https://www.kimi.com/`)。
2.  **打开“检查”工具**：
    * 在**聊天输入框**上点击鼠标右键。
    * 选择 **"检查"** (Inspect) 选项。
3.  **定位元素**：
    * 你的浏览器会打开一个开发者工具窗口，并高亮显示一行HTML代码，这行代码就是输入框（例如 `<div role="textbox" ...>`)。
4.  **复制选择器**：
    * 在**高亮的那行代码**上点击鼠标右键。
    * 选择 **复制 (Copy)** -> **复制选择器 (Copy selector)**。
    * 你会得到一个类似 `div[role="textbox"]` 或 `#app > ...` 的字符串。
5.  **更新插件设置**：
    * 回到本插件的“高级设置” -> “AI平台管理”。
    * 点击 "Kimi"。
    * 将你刚刚复制的字符串，粘贴到**"输入框CSS选择器"**字段中。
    * 点击 **"保存此AI"**。

**恭喜，你已经修复了插件！** 对豆包和通义千问也是同样的操作。

## ⚙️ 高级设置字段

### 🤖 AI平台管理
* **对话URL**: AI聊天页面的基础地址 (e.g., `https://www.kimi.com/`)
* **输入框CSS选择器**: 自动粘贴用的选择器 (e.g., `div[role="textbox"]`)
* **发送按钮CSS选择器**: 自动发送用的选择器 (e.g., `button[data-e2e="send-button"]`)

### 📝 提示词管理
* **可用变量**：`{platform}` `{url}` `{title}`

### 🌐 分析网站管理
* **匹配域名**: (e.g., `bilibili.com`)
* **标题CSS选择器**: (e.g., `h1.video-title`)
